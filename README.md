## How to use

### XFCE4:

```bash
pkill xfconf
rm -r ~/.config/xfce4
cp -r path/to/repo/xfce4 ~/.config/
```

### Alacritty

```bash
cp -r path/to/repo/alacritty ~/.config/
```

### Neovim

```bash
cp -r path/to/repo/nvim ~/.config/
```

### Fastfetch

```bash
cp -r path/to/repo/fastfetch ~/.config/
```
